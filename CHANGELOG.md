# Changelog

[Conventional Commit](https://www.conventionalcommits.org/) messages by
[commitizen](https://commitizen-tools.github.io/commitizen/) — do not edit this file by hand.

Regenerate or preview it with `make changelog` (or `bash tasks.sh changelog`). The published
docs site regenerates it fresh on every build; CI never commits it back to the repo.
